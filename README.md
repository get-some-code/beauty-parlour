# Habibs Hair and Beauty - Luxury Salon Digital Platform

A premium, agency-level digital product for a luxury beauty salon in New Town, Kolkata. This platform is designed to attract organic traffic, convert visitors into appointments, and build brand authority.

## 🌟 Premium Features

- **Luxury Hero Section**: High-impact visuals with elegant entrance animations.
- **Appointment Booking Flow**: A seamless 4-step process for service selection, scheduling, and confirmation.
- **Admin Dashboard**: A secure portal for the salon owner (Anamika) to manage bookings and customer data.
- **Advanced Local SEO**: Optimized for keywords like "best salon in New Town Kolkata" and "nail extensions New Town".
- **Responsive Design**: Flawless performance across mobile, tablet, and desktop devices.
- **Production Tech Stack**: Built with Next.js (App Router), TypeScript, Tailwind CSS, Framer Motion, and Supabase.

## 🎨 Design System

- **Champagne Beige (#E9DBBD)**: Main background for a warm, premium feel.
- **Soft Gold (#CBB682)**: Accents and glassmorphism cards.
- **Deep Bronze (#785F37)**: Primary call-to-action buttons.
- **Luxury Black (#181510)**: Strong contrast text and footer.
- **Typography**: Editorial serif for headings, modern sans-serif for body.

## 🛠️ Tech Stack

- **Frontend**: Next.js 14 (latest), React, TypeScript, Tailwind CSS.
- **Animations**: Framer Motion for smooth, expensive-feeling micro-interactions.
- **Icons**: Lucide Icons.
- **Backend/DB**: Supabase (Integration ready).
- **SEO**: Metadata API, JSON-LD Structured Data, Robots.txt, Sitemap.

## 🚀 Getting Started

1.  **Install dependencies**:
    ```bash
    npm install
    ```

2.  **Set up environment variables**:
    Create a `.env.local` file with your Supabase credentials:
    ```env
    NEXT_PUBLIC_SUPABASE_URL=your_supabase_url
    NEXT_PUBLIC_SUPABASE_ANON_KEY=your_supabase_anon_key
    ```

3.  **Run the development server**:
    ```bash
    npm run dev
    ```

4.  **Admin Login**:
    - URL: `/admin/login`
    - Demo Email: `admin@habibssalon.com`
    - Demo Password: `admin123`
